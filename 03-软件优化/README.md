# 💻 软件优化 · Software Optimization

> 面向 **高校实验与课题工程场景**，聚焦 Simulink 控制系统性能优化、线程管理与稳定性提升。

---

## 🧭 我在做什么？（一句话）

通过多线程多速率优化、CPU 核心绑定、能耗降低和安全设计，提高控制系统运行效率、稳定性与长时间运行能力（可保障 24 小时稳定运行）。

---

## 📂 技术聚焦

- CPU 与线程优化：核心绑定、多线程调用，降低 CPU 使用率
- 能耗控制：如18650 电池供电优化，实现长时间续航
- 稳定性设计：保障 24 小时稳定运行
- 安全设计：顶层保护策略（如限幅器器）避免硬件损坏
- 软件接口与扩展：支持硬件与第三方程序交互
